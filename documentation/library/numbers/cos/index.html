cos

calculate the cosine of given angle

Parameters

cos angle :integer :floating :complex

Returns

  • :floating
  • :complex

Examples

copy
print cos 0                 ; 1.0
print cos 0.3               ; 0.955336489125606
print cos 1.0               ; 0.5403023058681398

copy
cos to :complex [1 1]
; => 0.8337300251311491-0.9888977057628651i
